🏠 系統概覽 台股 / 美股分析平台

本平台是以 Python + Streamlit 建構的台股盤後量化分析系統， 整合 TWSE / TPEx / FinMind / yfinance 四大資料源，提供六因子量化選股、 三大法人籌碼、技術分析、基本面彙整、新聞情緒等功能， 並透過 GitHub Actions 自動執行每日盤後報告。

🧩 模組架構

各 Python 模組按職責分層。綠色 = 資料載入層，藍色 = 核心分析層，紫色 = 功能輔助層，橙色 = UI 主程式。

🟢 資料載入層
data_loader.py
TWSE/TPEx 市場行情載入、1小時快取、FALLBACK_DATA 備援
finmind_loader.py
FinMind API 月營收/季報/本益比；yfinance 備援；7天本機快取
news_analyzer.py
Google RSS / MoneyDJ / YouTube RSS 爬取、關鍵字情緒分析、1小時快取
us_market.py
yfinance 美股指數/關鍵股、台股供應鏈映射、1小時快取
macro_loader.py
WTI/美債/VIX/黃金/美元指數；yfinance；1小時快取
institutional_tracker.py
SEC 13F 大型基金持股追蹤（Berkshire / Bridgewater 等）
twse_institutional.py
三大法人買賣超、融資融券 — TWSE 全市場一次載入；1天快取
🔵 核心分析層
quant_model.py
六因子量化模型（動能/成長/品質/價值/籌碼/低波動）；跨截面 Z-score；機率預測
signal_engine.py
綜合技術+籌碼+基本面產生 BUY/SELL/HOLD 訊號
tech_analyzer.py
MA/RSI/MACD/KD/Bollinger/ATR/Fibonacci 技術指標與訊號
🟣 功能輔助層
portfolio.py
持倉管理 — 個股記錄、損益計算、持有天數
backtest.py
回測引擎 — 命中率、Brier Score、歷史驗證
utils.py
JSON 讀寫、HTTP 重試、資料轉換、SSL 修正
config.py
API 端點、路徑、因子權重、供應鏈樹、概念股、ETF 清單
📑 Tab 功能說明

12 個功能 Tab，涵蓋從總覽推薦到個股深度分析的完整流程。

Tab 0
🏆 整體分析
六因子量化 Top 10 推薦；市場概況（漲跌家數）；本週熱點話題；快速加入追蹤
Tab 1
🌍 美股連動
美股大盤指數（S&P500/Nasdaq/Dow）；台積電ADR；科技股與供應鏈連動分析
Tab 2
📋 候選清單
完整量化候選清單；多欄籌碼/技術/基本面信號；CSV 下載
Tab 3
🔍 個股分析
個股完整分析：行情/估值/技術圖/月營收趨勢/季報EPS表/PE歷史分位數/新聞情緒
Tab 4
💼 持倉管理
持倉新增/刪除；總損益統計；各股損益率、持有天數、策略建議
Tab 5
⭐ 追蹤清單
自選股追蹤；基本面+新聞自動載入；20日機率/期望報酬/信心度指標
Tab 6
🔥 熱度排行
產業熱度 Top 20 橫條圖；熱度分解（籌碼/技術/新聞/漲跌分）；成分股儀表板
Tab 7
📊 產業瀏覽器
概念股/供應鏈 pills 切換導覽；卡片行股票清單（漲紅跌綠）；摘要指標；產業新聞
Tab 8
⚙️ 模型設定
六因子權重調整（動能/成長/品質/價值/籌碼/低波動）；分數預覽；存檔
Tab 9
🎯 潛力股
落後補漲候選排行；peer_lag_score × 族群平均動能差；外資剛建倉訊號
Tab 10
📈 ETF 排行
台灣 ETF 月/季/年報酬率排行；ETF 詳情（前十大成分股）
Tab 11
📊 基本面彙整
自選股跨公司比較：月營收 YoY% 矩陣（正綠負紅）、近8季 EPS 矩陣、最新指標快照
🔄 資料流程

從外部 API 到 Streamlit UI 的完整資料流。每層都有本機快取以降低 API 呼叫頻率。

來源
TWSE OpenAPI TPEx OpenAPI FinMind API yfinance Google RSS SEC EDGAR
↓
載入/快取
data_loader → market_cache.json（1h） finmind_loader → finmind_cache.json（7d） news_analyzer → news_cache.json（1h） twse_institutional → institutional_cache.json（1d） us_market → us_market_cache.json（1h） macro_loader → macro_cache.json（1h）
↓
分析/計算
quant_model.enrich_dataframe() → 六因子 Z-score 評分 signal_engine.generate() → BUY/SELL/HOLD tech_analyzer.analyze() → MA/RSI/MACD/KD finmind_loader.get_financial_trend() → 季報 EPS 趨勢
↓
Streamlit
app.py → 12 Tab UI → 使用者瀏覽器
⚡ 快取策略：首次載入後資料保存本機 JSON，後續在 TTL 內直接讀快取，不重複打 API

🔌 API 依賴清單

平台使用 5 個外部資料來源，其中 TWSE/TPEx/Google RSS 完全免費無需帳號。

📈 TWSE OpenAPI
openapi.twse.com.tw/v1/
免費 即時

上市公司清單、日線行情、三大法人。盤中每 20 分鐘更新，盤後約 15:30 完整資料。

📊 TPEx OpenAPI
tpex.org.tw/openapi/v1/
免費 即時

上櫃公司清單與日線行情。與 TWSE 合併後可覆蓋全市場台股。

💹 FinMind API
api.finmindtrade.com/api/v4/data
需 Token 非即時

月營收、季報財務、本益比歷史。免費帳號每日可呼叫上限約 600 次；finmindtrade.com 免費申請。

📉 yfinance
finance.yahoo.com（非官方）
免費 延遲 15m

美股指數、個股基本面（PE/EPS/毛利率）、歷史 OHLCV。偶有限速；FinMind 資料優先，yfinance 作備援。

📰 Google RSS / MoneyDJ
news.google.com / moneydj.com
免費 近即時

台股財經新聞 RSS Feed。關鍵字過濾後做情緒分析（正面/中性/負面）；YouTube RSS 追蹤財經頻道。

📬 Gmail SMTP
smtp.gmail.com:587
免費

每日報告寄送用。需在 Google 帳號設定「應用程式密碼」；儲存在 GitHub Secret GMAIL_APP_PASSWORD。
