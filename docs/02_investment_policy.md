# 02 — 投資政策與平台原則

## 六大核心原則

1. **不追高**：股價明顯偏離基本面時主動提醒，不鼓勵買進
2. **不炒短線**：不提供當沖買賣點，不鼓勵頻繁交易
3. **長期穩健優先**：所有評分與建議以長期資產增值為目標
4. **第一性原則**：分析回到「公司如何賺錢、利潤是否可持續」
5. **事實查核**：不捏造財報數字、法人數據、目標價
6. **合規底線**：不提供內線消息、操縱市場建議或保證獲利承諾

## 禁止輸出清單

以下內容平台不得輸出：

- 明確當沖 / 隔日沖買賣點
- 「必漲」、「穩賺」、「翻倍」等保證語言
- 未揭露風險的個人化買賣指令
- 鼓勵高槓桿、融資追漲

## 必加風險提醒的情境

- 個股買賣建議
- 槓桿 ETF、期貨、選擇權
- 高波動小型股、生技股
- 財報不透明公司
- 任何要求費用的 API 或服務

## 標準免責聲明

> 本平台分析結果僅供研究與風險評估參考，不構成個人化投資建議。
> 實際投資前，請依照個人資金狀況、風險承受度與投資期限自行判斷，
> 必要時請諮詢合格財務顧問。歷史績效不代表未來報酬。
