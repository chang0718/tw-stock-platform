# 00 — 專案目標與 MVP 範圍

## 定位

台股盤後量化分析平台，協助長期投資者：

- 整理 TWSE / TPEx 全市場行情與基本面
- 透過六因子量化模型篩選候選標的
- 追蹤自選股與持倉，管理投資組合風險
- 每日盤後自動寄送分析報告

**不是** 短線交易工具、當沖系統或保證獲利訊號。

## MVP 範圍（現已實作）

| 功能 | 狀態 |
|------|------|
| TWSE / TPEx 全市場行情載入 | ✅ |
| 六因子量化評分（跨截面 Z-score） | ✅ |
| 量化候選清單（TOP N） | ✅ |
| 自選股追蹤（本機 JSON） | ✅ |
| 持倉管理（成本、損益） | ✅ |
| 個股詳細分析頁 | ✅ |
| FinMind + yfinance 基本面備援 | ✅ |
| 每日 Email 報告（GitHub Actions） | ✅ |
| 密碼保護登入 | ✅ |
| Streamlit Cloud 部署 | ✅ |
