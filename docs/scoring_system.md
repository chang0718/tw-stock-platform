# 台股量化平台 — 選股評分架構說明

## 一、六大因子分數（各 0–100 分）

| 因子 | 預設權重 | 數據來源 | 計算方式 | 學術依據 |
|---|---|---|---|---|
| **動能** momentum | 25% | 本機歷史快照 | MA20 乖離(60%) + MA60 乖離(40%) | Jegadeesh-Titman 1993 |
| **成長** growth | 20% | FinMind API | 營收 YoY + EPS 成長率 平均 | 盈餘動能文獻 |
| **品質** quality | 20% | FinMind API | 毛利率 + 淨利率 + EPS 成長 平均 | Novy-Marx 2013 |
| **價值** value | 15% | FinMind API | E/P = 1/PE（越便宜分數越高） | Fama-French 1992 |
| **籌碼** flow | 10% | TWSE T86 | 外資淨買超（千股） | Gompers-Metrick 2001 |
| **低波動** low_vol | 10% | 本機歷史快照 | 20 日波動率（越低越好） | Frazzini-Pedersen 2014 |

> **重要**：六大因子全部用**跨截面 Z-score 標準化**（和當日全市場所有股票比較排名），不是絕對分。
> 無基本面數據的股票，成長／品質／價值固定給中性 50 分，防止假訊號。

---

## 二、評分路徑（依資料完整度分兩種）

| 路徑 | 條件 | 複合分計算方式 |
|---|---|---|
| **完整路徑** | 有 FinMind 基本面 | 六因子加權（依上表權重） |
| **技術動能路徑** | 無基本面（未加入追蹤清單） | 動能 45% + 籌碼 30% + 低波動 25%（技術 60% + 原始 40% 混合） |

---

## 三、複合分 → 上漲機率

| 時間週期 | Horizon 調整 | 說明 |
|---|---|---|
| prob5（5 日） | −0.08 | 短期反轉傾向，微降 |
| prob20（20 日） | 0 | 基準，主要選股依據 |
| prob60（60 日） | +0.10 | 中期動能持續，微升 |

**計算公式：**

```
z = (composite_score - 50) / 15 + horizon_adj - risk_penalty
prob = Φ(z) × 100，範圍限制在 10–90%
```

- `risk_penalty`：風險分 > 70 時 −0.15，> 55 時 −0.10，其餘 0
- `Φ`：標準常態分佈 CDF（使用 Acklam 有理近似，不依賴 scipy）

## 四、候選等級判定

| 等級 | prob20 | 信心度 | 風險分 |
|---|---|---|---|
| **核心候選** | ≥ 60% | ≥ 55 | ≤ 70 |
| **觀察候選** | ≥ 55% | ≥ 45 | ≤ 80 |
| **高風險觀察** | ≥ 58% | — | ≥ 70 |
| **保守觀望** | 其餘 | — | — |

## 五、最終排名分（final_composite）

```
final_composite =
  prob20        × 35%
+ 信心度        × 20%
+ (prob20×85%) × 20%   ← 命中率佔位（快照累積後改用真實命中率）
+ (100 - 風險) × 15%
+ composite    × 10%
+ 板塊偏好加成  +4 分   ← 使用者選的「特別關注產業」
+ 資料完整加成  +3 分   ← 有基本面 + 籌碼 + 技術三者齊全
```

## 六、資料完整度對評分的影響

| 資料狀態 | 影響 |
|---|---|
| 有 FinMind 基本面 | 成長／品質／價值因子使用真實數據 |
| 有 TWSE 三大法人 | 籌碼因子使用真實數據 |
| 有 20 天以上本機快照 | 動能／低波動因子使用真實數據 |
| 三者都有 | `complete_score = True`，+3 分加成，排名優先 |
| 都沒有 | 六因子全給 50（中性），不產生假訊號 |

---

## 七、如何讓評分更準確

1. **每日盤後點「載入上市」或「載入全市場」**
   - 累積本機歷史快照，動能和低波動因子才有真實數據

2. **將感興趣的股票加入追蹤清單**
   - 系統自動從 FinMind 載入基本面，解鎖成長／品質／價值三個因子

3. **每日點「保存今日快照」**
   - 累積 10 天快照後，命中率統計從佔位值改為真實歷史勝率

4. **調整「特別關注產業」**
   - 在側邊欄選擇偏好板塊，該產業股票 final_composite 加 4 分
