## C. 評估方法合理性（對照財金學界實務）

紙上交易 `equity_metrics` 已依 CLAUDE.md 回測規範與主流量化教科書設計：

| 指標 | 意義 | 學界依據 |
|------|------|----------|
| 累積/年化報酬 | 整體績效 | — |
| **Alpha (vs 0050)** | 扣除大盤後的超額報酬（是否有選股價值）| CAPM / 主動報酬 |
| **Sharpe** | 風險調整後報酬（每單位波動的超額報酬）| Sharpe (1966) |
| **最大回撤 MDD** | 下檔風險（最慘跌幅）| 風險控管標準 |
| 勝率 | 已平倉交易獲利比例 | — |
| **IC 資訊係數** | 進場分數與實現報酬的 Spearman 等級相關 | Grinold & Kahn《Active Portfolio Management》— 衡量選股「訊號 → 報酬」預測力的標準量 |

**合規/嚴謹性保證**：
- 計入交易成本（手續費 0.1425%）、賣出證交稅 0.3%、滑價 0.1% → 不高估績效。
- **不使用未來資料**：每日僅用當日盤後分數決策，下一交易日才 mark-to-market 結算。
- 比較基準（0050），標示最大回撤，全頁附「歷史回測不代表未來報酬」免責。
- 同日重跑冪等（不重複下單），帳本 atomic 寫入。

**尚待強化（v2）**：
- **survivorship bias**：目前 universe 為當前上市櫃清單；長期累積時若有下市股，需保留
  其歷史以免倖存者偏誤（現為 forward 累積、暫無此問題，但回補歷史回測時須注意）。
- IC 目前用「已平倉交易」計算，樣本較慢累積；可另加「每日 rank IC」（當日分數 vs
  次日報酬的橫斷面相關）更快取得統計檢定力。
- 加入基準風險調整後的 **Information Ratio**（Alpha / 追蹤誤差）。
